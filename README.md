This repository was used to do all the experiments in the paper, 
To reproduce the results , do the following things 
1) Setup the SLM using configuration manager and follow all the details mentioned in the manual 
2) Add appropriate DLLs in the folders mentioned
3) Uncomment the appropriate train test set in the pelm/config according to the dataset
4) Run the main.py using the python 3.11 (as newer environments were not sutaible for that camera model)
5) Make sure to delete or transfer the npz files to prevent overwriting
